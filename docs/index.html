Tailviewer

A fast, free and open source log viewer for Windows

Welcome Tailviewer.

Tailviewer is a log file viewer for Windows that allows you to view text based log files both offline and in real-time. It is easy to use, incredibly fast and free to use.

Features

Tailviewer can display any text based log file and allows tailing, searching and filtering.

Notifications

Tailviewer automatically notifies you when an opened log file is modified so you never miss a thing!

Filtering

You can perform sub string, wildcard and regular expression searches through log files.
